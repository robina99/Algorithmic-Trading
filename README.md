# Algorithmic-Trading
This project investigates and compares multiple algorithmic trading approaches for predicting stock price movements and generating profitable trading strategies. The study evaluates both evolutionary computation techniques and deep learning recurrent neural network architectures using historical Google (GOOG) stock market data from 2014 to 2020.

The implemented models include:

Genetic Algorithm (GA)
Genetic Programming (GP)
Vanilla Recurrent Neural Network (RNN)
Long Short-Term Memory Network (LSTM)
Gated Recurrent Unit (GRU)

The objective of the project is to determine which model provides the best balance between predictive accuracy, profitability, and trading stability when applied to algorithmic trading.
